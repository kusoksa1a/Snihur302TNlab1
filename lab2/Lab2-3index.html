Програмування

Одним з моїх улюблених хобі є програмування. Зараз я вивчаю мову програмування Python. Ця мова дозволяє створювати потужні програми для різних сфер діяльності.


length = float(input("Введіть довжину прямокутника: "))
width = float(input("Введіть ширину прямокутника: "))

area = length * width

print("Площа прямокутника дорівнює", area)
